# base-sdk
> 基础SDK

- [Vue.js](https://cn.vuejs.org/)
- [element-ui](http://element.eleme.io/#/zh-CN)

## 产出
- [base-sdk](http://base-sdk.cancss.com/#/)

## 复制配置文件

复制`server.config.example.js`为`server.config.js`（此文件会被git忽略），用于配置代理目标host。

## 安装依赖

```
npm install
```

## 启动项目

```
npm run serve
```

## 部署项目

```
npm run deploy
```
